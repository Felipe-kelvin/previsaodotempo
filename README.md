# 🌦️ Previsão do Tempo

Aplicação web que permite consultar a previsão do tempo de qualquer cidade, com suporte a busca por voz.

## 🚀 Funcionalidades

- 🔍 Busca de cidade
- 🌡️ Exibição de temperatura em tempo real
- 💧 Umidade do ar
- 🌥️ Ícones climáticos dinâmicos
- 🎤 Reconhecimento de voz

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript
- API OpenWeatherMap
